# Timetable_Django

 This project is written in Python with Django framework.
## Feature summary
- Edit Schedule 
- Add/Remove Task for each Subject
- Show Task everyday

## Frameworks, libraries,...
- Front-end: Bootstrap
- Back-end: 
  - Django
  - Database: SQLite

## Setting

To run this Project, use python cmd:

For Window
```
py manage.py runserver
```
For MAC/Linux
```
python3 manage.py runserver
```
